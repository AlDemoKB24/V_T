## Краткое описание работы со страницой:
### В поле ввода нужно ввести задачу, после чего при нажатии на "+" задача будет добавлена в список задач снизу. Задачу можно отметить как выполненную, можно отменить эту отметку, можно удалить задачу. Задачи можно фильтровать по критерию завершённости. Можно менять задачи местами. Для этого нужно, зажав ЛКМ на какой-то из задач, перетащить её на другую задачу и отпустить ЛКМ.
